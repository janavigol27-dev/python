#### **Retail Transaction Analysis \& Dashboard Project**



* **Project Overview**



This project focuses on analyzing retail transaction data to gain insights into sales performance, customer behavior, and regional trends. The project involves two main parts:

SQL Database Management: Cleaning and querying 500+ transaction records.

Power BI Dashboard: Visualizing key performance indicators (KPIs) for real-time business monitoring.



* **Key Data Steps Taken**



Data Collection: Created a dataset of 500 retail transactions including Date, Product, Category, Region, and Payment Mode.

SQL Implementation: Developed a robust SQL script to create the database, define table structures, and insert large-scale data.

Data Cleaning: Handled missing values and ensured correct data types for dates and currency fields.



* **Important SQL Queries \& Logic**



I implemented several complex queries to extract deep business insights:

Total Revenue Calculation: Summarizing the total sales amount across all categories.

Monthly Trends: Extracting month-wise sales to identify peak performance periods.

Customer Segmentation: Identifying high-value customers based on transaction frequency.

Regional Analysis: Ranking regions by total sales to optimize marketing efforts.



* **Dashboard Features \& KPIs**



The Power BI dashboard provides a visual summary of the following metrics:

Sales KPIs: Total Sales, Average Order Value (AOV), and Growth percentage vs. previous period.

Customer Insights: Real-time tracking of New vs. Returning customers.

Product Performance: Top 5 best-selling products visualized through bar charts.

Category Share: A donut chart showing sales distribution across Electronics, Clothing, and more.



* **Key Business Insights**



Growth: Observed an 18.6% increase in total sales compared to the last period.

Payment Preference: UPI is the most popular payment method, used in over 45% of transactions.

Top Region: The South \& West regions contribute the highest revenue share.

Customer Loyalty: High retention rate identified through the Returning Customers KPI.



* **Tools Used**



SQL (MySQL/PostgreSQL): For data storage and complex querying.

Power BI: For interactive visualization and DAX measure calculation.

Excel: For initial data structuring and cleaning.
